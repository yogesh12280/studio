# **App Name**: OrgaBlast

## Core Features:

- Bulletin Creation: Allow users to create and post bulletins with text, images, and links. Users can post responses, like the post, and show the total count of viewers.
- Employee Categorization: Categorize employees into different departments, teams, or roles for targeted bulletin delivery.
- Targeted Delivery: AI-powered tool to allow admins to select specific categories of employees to receive certain bulletins. The tool will analyze employee roles and departments, recommending optimal groupings for efficient information dissemination.
- Bulletin Scheduling: Schedule bulletins to be posted at a future date and time. Display the bulletin category wise like organization and employee.
- Search and Filter: Enable users to search and filter bulletins by keywords, categories, and date.
- User roles: Define user roles, to separate the rights of bulletin creation/editing from bulletin consumption. Remove the post based on end date or creator can delete it.

## Style Guidelines:

- Primary color: Light indigo (#7986CB), a calm but professional color.
- Background color: Very light indigo (#F2F3F8), close to white.
- Accent color: Soft violet (#9FA8DA) to subtly call attention to a limited number of on-screen elements.
- Headline and body font: 'PT Sans', a versatile sans-serif suitable for professional communication.
- Use clear, consistent icons for bulletin categories and employee roles.
- A clean, organized layout with a clear hierarchy for easy navigation and information consumption.
- Subtle animations for transitions and notifications to enhance the user experience without being distracting.
